# TabView & Navigation Demo / Demonstração de TabView e Navegação

> **Languages / Idiomas:** [Português](#português) | [English](#english)

---

<a name="português"></a>
## 🇧🇷 Português

### 📝 Descrição do Projeto
Este projeto é uma aplicação iOS desenvolvida em **SwiftUI** para demonstrar o uso de navegação por abas utilizando o componente `TabView` combinado com `NavigationStack`.

O aplicativo possui 4 abas principais na barra inferior:
1. **Rosa (`TelaRosa`):** Exibe um fundo rosa personalizado com um círculo central e o ícone de pincel (`paintbrush`).
2. **Azul (`TelaAzul`):** Exibe um fundo azul com um círculo central e o ícone de pincel pontiagudo (`paintbrush.pointed`).
3. **Cinza (`TelaCinza`):** Exibe um fundo cinza com um círculo central e o ícone de paleta de pintura (`paintpalette`).
4. **Lista (`TelaLista`):** Exibe uma `List` com itens estilizados e ícones associados aos temas das abas anteriores.

---

### 🛠️ Tecnologias Utilizadas
- **Linguagem:** Swift
- **Framework:** SwiftUI (`TabView`, `NavigationStack`, `List`, `SF Symbols`)
- **IDE:** Xcode
- **Plataforma Target:** iOS / iPadOS

---

### 🚀 Como Executar o Projeto

#### Pré-requisitos
* Um computador Mac rodando **macOS**.
* **Xcode 14** ou superior instalado.

#### Passo a Passo
1. **Clonar ou Baixar o Repositório:**
   ```bash
   git clone <URL_DO_REPOSITORIO>
   ```
2. **Abrir o Projeto no Xcode:**
   * Navegue até a pasta do projeto.
   * Dê um duplo clique no arquivo `.xcodeproj` (ex: `Desafio3 - 29-05.xcodeproj`).
3. **Verificar Ativos (Assets):**
   * Certifique-se de que a cor personalizada `rosa` está definida em `Assets.xcassets`.
4. **Executar no Simulador:**
   * Selecione o simulador desejado na barra superior (ex: *iPhone 15*).
   * Pressione `Cmd + R` ou clique no botão **Play (▶)** para compilar e executar o aplicativo.

<a name="english"></a>
## 🇬🇧 English

### 📝 Project Description
This is an iOS application developed in **SwiftUI** showcasing tab-based navigation using the `TabView` component enclosed within a `NavigationStack`.

The app contains 4 main tabs in the bottom bar:
1. **Pink (`TelaRosa`):** Displays a custom pink background with a centered circle and a `paintbrush` SF Symbol.
2. **Blue (`TelaAzul`):** Displays a blue background with a centered circle and a `paintbrush.pointed` SF Symbol.
3. **Gray (`TelaCinza`):** Displays a gray background with a centered circle and a `paintpalette` SF Symbol.
4. **List (`TelaLista`):** Displays a `List` with enumerated items alongside their corresponding paint-themed icons.

---

### 🛠️ Tech Stack
- **Language:** Swift
- **Framework:** SwiftUI (`TabView`, `NavigationStack`, `List`, `SF Symbols`)
- **IDE:** Xcode
- **Target Platform:** iOS / iPadOS

---

### 🚀 How to Run the Project

#### Prerequisites
* A Mac computer running **macOS**.
* **Xcode 14** or higher installed.

#### Step-by-Step
1. **Clone or Download the Repository:**
   ```bash
   git clone <REPOSITORY_URL>
   ```
2. **Open the Project in Xcode:**
   * Navigate to the project folder.
   * Double-click the `.xcodeproj` file (e.g., `Desafio3 - 29-05.xcodeproj`).
3. **Verify Color Assets:**
   * Make sure the custom named color `rosa` exists in `Assets.xcassets`.
4. **Run in Simulator:**
   * Select your target simulator from the top toolbar (e.g., *iPhone 15*).
   * Press `Cmd + R` or click the **Play (▶)** button to build and run.
